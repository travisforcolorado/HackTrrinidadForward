# Trinidad Help Finder SPA

This project hosts an accessible single page application that highlights health, housing, food, and heat resources for residents in and around Trinidad, Colorado.

## Getting started

```bash
npm install
npm run dev
```

The application is built with [Vite](https://vitejs.dev/) and React. Run `npm run build` to produce the static assets you can deploy to GitHub Pages. See [DEPLOYMENT.md](./DEPLOYMENT.md) for detailed deployment instructions.
